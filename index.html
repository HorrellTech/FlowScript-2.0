Getting Started

Welcome to FlowScript 2.0! This visual website builder lets you create HTML, CSS, and JavaScript through an intuitive drag-and-drop interface.

Quick Start Guide

  1. Choose a Template: Select from pre-built templates or start with a blank page
  2. Drag Blocks: Drag blocks from the toolbox (left panel) to the workspace (center)
  3. Configure Blocks: Fill in the input fields for each block
  4. Preview: Click the "Preview" button to see your website in action
  5. Export: Save your work or export the final HTML
Tip: Start with the "Landing Page" template to see FlowScript in action!

Interface Overview

FlowScript's interface is divided into three main areas:

1. Toolbox (Left Panel)

  • HTML Blocks: Document structure, headings, paragraphs, images, links
  • CSS Blocks: Style rules, properties, custom CSS
  • JavaScript Blocks: Variables, functions, event listeners, loops
  • Custom Blocks: Advanced and fallback elements

2. Workspace (Center Panel)

  • Main building area where you construct your website
  • Drag blocks here from the toolbox
  • Nested drop zones for structured content (like head/body sections)
  • Each block can be configured with input fields

3. Output Panel (Right Panel)

  • Shows the generated HTML/CSS/JavaScript code in real-time
  • Copy Code: Copy the generated code to clipboard
  • Preview: Open your website in a new window

Header Controls

  • File Management: New, Open, Save, Export buttons
  • Theme Selector: Choose from 9 built-in themes
  • Template Selector: Load pre-built templates
  • Mobile Toggle: Switch to mobile-friendly interface

Working with Blocks

Adding Blocks

  • Drag & Drop: Drag any block from the toolbox to the workspace
  • Quick Add: Click the "+" button on any block to add it instantly
  • Nested Placement: Drop blocks into nested zones (like head/body sections)

Configuring Blocks

  • Each block has input fields for customization
  • Fill in text fields, select options, and configure properties
  • Changes are reflected in the generated code immediately
  • Use the delete button (×) to remove unwanted blocks

Block Categories

HTML Blocks

Structure your webpage with HTML elements like headings, paragraphs, images, and containers.

CSS Blocks

Style your webpage with CSS rules, properties, and custom styling.

JavaScript Blocks

Add interactivity with variables, functions, event listeners, and logic.

Custom Blocks

Advanced blocks for custom HTML elements and raw code.

File Management

Saving Your Work

  • Save Workspace: Saves your current blocks as a JSON file that can be reopened later
  • Export HTML: Exports the generated code as a standalone HTML file
  • Both options preserve your work for future editing or deployment

Opening Files

  • JSON Files: Reopens saved FlowScript workspaces
  • HTML Files: Imports existing HTML and converts it to blocks (when possible)
  • The import feature works best with clean, well-structured HTML

Starting Fresh

  • New: Clears the workspace and creates a basic HTML structure
  • Includes essential elements like DOCTYPE, head, and body sections
  • Perfect starting point for new projects

Mobile Mode

FlowScript automatically adapts to mobile devices and small screens, but you can also manually toggle mobile mode.

Mobile Features

  • Panel Toggles: Left and right buttons to show/hide toolbox and output panels
  • Mobile Menu: Dropdown menu for file management, themes, and templates
  • Touch-Friendly: Larger touch targets and improved interactions
  • Auto-Detection: Automatically enables on screens ≤ 768px wide

Mobile Controls

  • Toolbox Toggle: Left panel button or Ctrl/Cmd + T
  • Output Toggle: Right panel button or Ctrl/Cmd + O
  • Mobile Menu: Hamburger menu or Ctrl/Cmd + M
  • Close Panels: Tap overlay or press Escape
Mobile Tip: Use the "+" buttons on blocks for quick addition on touch devices!

AI Assistant

The AI Assistant helps you generate FlowScript blocks and complete HTML/CSS/JS code using advanced AI models like OpenAI GPT-4, Claude, Gemini, and Cohere.

How to Use the AI Assistant

  1. Configure API Keys: Go to the Settings tab in the AI modal and enter your API key for your chosen provider.
  2. Select Provider & Model: Choose your AI provider and model in the Generate tab.
  3. Write a Prompt: Describe what you want to build. The more specific, the better!
  4. Generate: Click Generate FlowScript and wait for the AI to produce code.
  5. Review & Use: Review the generated code and click Use This Code to load it into your workspace.

Example Prompt

Copy & Edit: 
Create a modern landing page for a fitness app. 
Include a hero section with a headline and call-to-action button, 
a features list with icons, and a contact form. 
Use a blue and green color scheme, and make the design responsive.

Tips for Better Results

  • Be specific about layout, colors, and features you want.
  • Mention if you want CSS styling or JavaScript functionality.
  • Review and edit the generated code as needed.
  • Try different providers/models for varied results.

Tips & Tricks

Building Efficiently

  • Start with Structure: Begin with an HTML Document block, then add head and body content
  • Use Templates: Start with a template close to your desired design
  • Group Related Blocks: Keep CSS and JavaScript blocks organized near their related HTML
  • Preview Often: Use the preview button to see how your site looks

Advanced Techniques

  • Custom CSS: Use Raw CSS blocks for advanced styling not covered by other blocks
  • JavaScript Interactivity: Combine Get Element, Add Event Listener, and function blocks for dynamic behavior
  • Nested Containers: Use Div Container blocks to create complex layouts
  • Import Existing Code: Import HTML files to convert existing websites to blocks

Troubleshooting

  • Missing Content: Check that blocks are in the correct nested zones (head vs body)
  • Styling Issues: Ensure CSS blocks are in the head section or use inline styles
  • JavaScript Errors: Use Console Log blocks to debug your JavaScript
  • Import Problems: Complex HTML may not import perfectly - use custom blocks as needed

Keyboard Shortcuts

  • Ctrl/Cmd + S: Quick save (triggers save workspace)
  • Ctrl/Cmd + N: New workspace
  • Ctrl/Cmd + O: Open file
  • Escape: Close modals and mobile panels
Pro Tip: Save your workspace frequently and experiment with different themes to find the perfect look!
